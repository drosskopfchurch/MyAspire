## Steps 
1. Download and install FluentUI Templates
```dotnet new install Microsoft.FluentUI.AspNetCore.Templates```
2. Install Vs Code
3. Install C# Dev Kit
4. Debug Application

    - Choose <ProjectName> 
    - Choose <ProjectName>.AppHost.csproj
